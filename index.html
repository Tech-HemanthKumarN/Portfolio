Vacation Rental Marketplace

Vacation rental marketplace designed to connect property owners with travelers seeking short-term accomodations. This platform allow the host to easily create listings with location and photos, while guests can browse, book, and leave reviews. It also have user login authentication system to make everything secure and easy to manage for both hosts and guests. The main goal is to make booking vacation rentals quick and simple. To build This project i used NodeJs and Express for backend to handle routing and server side logic. I integrated MongoDB Atlas as a Cloud database. And Passport for authentication, Cloudinary to handle and serve images efficiently.
